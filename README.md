# Homeet: Поиск соседей
<h2>Краткий обзор проекта.</h2>

<h2>Функционал:</h2>
<h3>1. Создание анкеты<br></h3>
<b>Пользователю выдается краткая информация по боту(подробнее доступно по /help) и предлагается создание анкеты, в которой нужно следующие данные:<br></b>
В любой момент пользователь может либо отменить создание анкеты /cancel либо вернуться на один вопрос назад /back<br><br>
- Имя<br>
- Возраст<br>
- О себе<br>
- О соседе(Описывает какого соседа хочет найти)<br>
- Пол<br>
- Курс<br>
- Ссылки на соцсети<br>
- Аватарка<br>
- Желаемый пол соседа<br>
- Желаемый курс соседа<br><br>
<b>Далее разветлвение, пользователь либо ищет квартиру, либо хочет найти соседа в свою квартиру</b><br>
Искателю квартиры показываются анкеты владельцев квартир, а так же анкеты таких же искателей, чтобы объединиться в поиске квартиры<br>
Владельцу квартиры показываются анкеты искателей квартиры<br><br>
Ищет квартиру:<br>
- Максимальная стоимость квартиры<br>
- Рядом с какими станциями метро ищет квартиру<br><br>
Сдает квартиру:<br>
- Цена квартиры<br>
- Станция метро квартиры<br>
- Точный адрес квартиры<br>
- Время до корпусов вышки<br>
- Описание квартиры<br>
- Фотографии квартиры<br><br>

Весь пользовательский ввод, который может повлиять на работу бота проверяется<br>

<h3>2. Действия с анкетой</h3>
<b>- Посмотреть анкету:</b> Пользователю выводится анкета в том виде, в котором она будет показываться остальным пользователям<br><br>
<b>- Изменить анкету:</b> Выводится вся информация из профиля пользователя, а так же анкета как в пункте "посмотреть анкету",
пользователь может изменить любые данные, критерии поиска, фотографии и тд. Нельзя поменять пункт что ищет пользователь (Квартиру или Соседа в свою квартиру)
т.к это влияет на уже сформировавшиеся мэтчи. Пользовательский ввод так же проверяется<br><br>
<b>- Удалить анкету:</b> Удаляется анкета пользователя, включая все упоминания о нем в полях других пользователей<br><br>
<h3>3. Matche`s</h3>
<b> - Смотреть анкеты:</b> Предлагаются анкеты пользователей, которые пользователь еще не видел, по одной, есть возможность поставить лайк либо дизлайк, если новых анкет нет, можно удалить историю просмотров и смотреть все анкеты заново<br><br>
<b> - Смотреть мэтчи:</b> Выводится количество взаимных лайков у пользователя и предлагается посмотреть их, при просмотре выводится анкета пользователя, с котороым случился мэтч, а так же его контакты<br><br>
Как работает: Когда пользователь ставит лайк анкете, код проверяет стоит ли лайк у владельца этой анкеты на пользователя, если стоит, обоим приходит уведомление о мэтче и сам мэтч сохраняется в 'мои мэтчи'
